Player jQuery plugin

Player jQuery plugin provides a common interface to Youtube and Vimeo Player API.

HTML

<script src="/js/player.js"></script>

Javascript

$('.player').player({url:'http://www.youtube.com/watch?v=yb512wSjyVs'});
